# Bird to Nest

A small static HTML/CSS/JavaScript canvas game.

Help the bird climb from the forest floor to its nest while dodging branches and eagles, rescuing companions, and using boost carefully.

## Play

Open `index.html` in a browser, or publish the folder with GitHub Pages.

## Controls

- Move: Arrow keys or WASD
- Boost: Space, ArrowUp, or W
- Rescue: E, tap a nearby cage, or tap Rescue
- Restart: R
- Touch: on-screen left, boost, right, and rescue buttons
